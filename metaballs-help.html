Back to Metaballs

Getting Started

  • Add primitives: Drag shapes from the tray into your scene.
  • Move, scale, rotate: Use standard visionOS manipulation gestures to adjust primitives.
  • Edit materials: Tap a metaball to open its material settings.
  • Set defaults: Tap the palette icon in the tray to choose the material for new primitives.
  • Positive & negative: Use the +/- toggle to switch a metaball between additive (positive) and subtractive (negative).
  • Metaballs+: Some shapes and materials are part of the Metaballs+ subscription. Think of it as a way to support indie app development and future updates to this playful tool.

Saving Your Work

To save the current version of your scene, tap the gallery icon in the toolbar. This will bring up the gallery as well as save the current state of your scene.

Please note that if you work on a scene and leave the app, the scene does not 'autosave' at this time. We may change this behavior in future versions - but for now, we've opted to have this be a manual step.

Export

Metaballs v1.2 now supports USDZ and STL export!

  • USDZ: Universal format that's viewable on device with the same materials as in the app, and can be brought into desktop content creation software for further exploration.
  • STL: Common 3D printing format. (Not viewable on device.)

If you do not see these two export options, please make sure you're updated to the latest version of the app.
